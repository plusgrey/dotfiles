# Tmux 快捷键速查表

## 基本设置

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 前缀键 | `Ctrl + b` | 所有命令的前导键 |
| 重载配置 | `Prefix + Ctrl + r` | 重新加载 ~/.tmux.conf |

## 会话管理

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 新建会话 | `Prefix + Ctrl + c` | 创建新会话 |
| 新建会话 | `Alt + Shift + S` | 通过脚本创建新会话 |
| 重命名会话 | `Prefix + .` | 重命名当前会话 |
| 选择会话树 | `Prefix + W` | 选择会话/窗口树 |
| 移动会话 | `Prefix + H / L` | 向左/右移动会话顺序 |
| 切换会话 1-9 | `Ctrl + 1~9` 或 `F1~F5` | 快速切换到第 N 个会话 |

## 窗口管理

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 新建窗口 | `Alt + o` | 在当前路径创建新窗口 |
| 关闭窗格 | `Alt + Shift + Q` | 关闭当前窗格 |
| 重命名窗口 | `Prefix + ,` | 重命名当前窗口 |
| 选择窗口 1-9 | `Alt + 1~9` | 快速切换到第 N 个窗口 |
| 上一个窗口 | `Prefix + Ctrl + p` | 切换到上一个窗口 |
| 下一个窗口 | `Prefix + Ctrl + n` | 切换到下一个窗口 |
| 移动窗口位置 | `Alt + Shift + H / L` | 向左/右移动窗口 |
| 分离窗格 | `Alt + Shift + O` | 将当前窗格分离为新窗口 |
| 移动窗格到会话 | `Prefix + 1~9/0` | 将当前窗口移到第 N 个会话 |
| 合并窗格到窗口 | `Alt + Shift + 1~9` | 将当前窗格合并到第 N 个窗口 |

## 窗格分割

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 向上分割 | `Prefix + k` | 在当前窗格上方创建新窗格 |
| 向下分割 | `Prefix + j` | 在当前窗格下方创建新窗格 |

| 向右分割 | `Prefix + l` | 在当前窗格右侧创建新窗格 |

| 功能 | 快捷键 | 说明 |
|------|--------|------|

| 开始选择 | `v` | （复制模式中）开始选择文本 |

## 说明

- **Prefix** = `Ctrl + b`
- 所有 Alt 组合键可以直接使用，无需前缀
- 配置基于 Vim 键位 (hjkl)，适合 Vim 用户
- 窗口和窗格索引从 1 开始
- 会话通过管理脚本实现 1-based 编号系统

## 快速记忆技巧

### Vim 方向键记忆 (hjkl)
```
    k (上)
    ↑
h ← → l
    ↓
    j (下)
```

### 常用组合
- **导航**: `Alt + hjkl` - 切换窗格
- **调整**: `Alt + Shift + HJKL` - 调整窗格大小
- **分割**: `Prefix + hjkl` - 分割窗格
- **窗口**: `Alt + 1~9` - 切换窗口
- **会话**: `Ctrl + 1~9` - 切换会话
